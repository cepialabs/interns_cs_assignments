# Target Scope

## Target Name
Alibaba Group Holding Limited

## Primary Domain
alibaba.com

## Objective
The objective of this reconnaissance task is to understand the public-facing
attack surface of the target using passive reconnaissance techniques only.

## In-Scope Assets
- Main domain: alibaba.com
- Public subdomains of alibaba.com
- Public DNS records
- Publicly accessible web services (HTTP/HTTPS)
- Information available via OSINT sources

## Out-of-Scope Assets
- Any login-protected functionality
- Brute force attacks
- Exploitation of vulnerabilities
- Denial of Service (DoS/DDoS)
- Internal networks or private IPs
- Source code or database access

## Reconnaissance Type
Passive Reconnaissance (No active exploitation performed)

## Legal & Ethical Note
This assessment is performed strictly for educational purposes as part of
cybersecurity training. No harmful activity was conducted against the target.
