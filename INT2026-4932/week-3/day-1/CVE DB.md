1. Apache (Log4j Vulnerability)

What the vulnerability is:
A security flaw in Apache Log4j that allows hackers to run malicious code.

What impact it has:

System can be hacked

Data can be stolen

Why it matters:
Apache is used in many applications, so this vulnerability affects many systems.

2. SSH Vulnerability

What the vulnerability is:
Allows attackers to find valid usernames on a server.

What impact it has:

Helps attackers guess passwords

Increases risk of hacking

Why it matters:
Knowing usernames makes attacks easier.

3. MySQL Vulnerability

What the vulnerability is:
Allows users with limited access to get higher permissions.

What impact it has:

Unauthorized access to database

Data can be changed or stolen

Why it matters:
Databases store important and sensitive data.
