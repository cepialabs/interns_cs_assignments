# Week 1 Cybersecurity Basics

## 1. What cybersecurity professionals do
Red Team: Attacks systems to find weaknesses  
Blue Team: Defends and monitors systems  
Purple Team: Combines both teams for improvement  

## 2. Ethical hacking vs Illegal hacking
Ethical hacking is legal and done with permission  
Illegal hacking is unauthorized and punishable by law  

## 3. Responsible disclosure
Security researchers privately inform companies about vulnerabilities before public release  

## 4. Why isolated labs are mandatory
Prevents accidental attacks on real systems  
Provides safe environment for testing tools  

## 5. Virtualization basics
Host OS = your real computer OS  
Guest OS = virtual OS inside software like VirtualBox  

## 6. Kali Linux ecosystem
Kali is a penetration testing OS with tools like:
Nmap, Wireshark, Metasploit, Burp Suite
