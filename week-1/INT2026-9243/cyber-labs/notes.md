# Linux Commands Notes

1. pwd – Shows current directory
2. ls – Lists files and folders
3. cd – Changes directory
4. mkdir – Creates a new directory
5. rmdir – Removes an empty directory
6. touch – Creates an empty file
7. rm – Deletes files or folders
8. cp – Copies files or directories
9. mv – Moves or renames files
10. cat – Displays file content
11. nano – Opens text editor
12. chmod – Changes file permissions
13. chown – Changes file ownership
14. ifconfig – Shows network configuration
15. ip a – Displays IP address information
